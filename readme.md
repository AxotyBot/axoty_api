# Axotiy API
   
The open API for the axoty discord bot.
    
## Get Request

For Images
´http://api.axoty.xyz/random?type=image´

For Videos
´http://api.axoty.xyz/random?type=videos´

For Facts
´http://api.axoty.xyz/random?type=facts´

## Get Response

    {
        string url
        string source
        string suggester
    
    }
